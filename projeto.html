Lazy Moon

Projeto

  O projeto "Lazy Moon" tem como objetivo criar um sistema de monitoramento e suporte remoto a uma rede LAN, utilizando-se de diversos programas criados pelos integrantes. O projeto consiste em um sistema de amplo espectro voltado ao monitoramento e gerenciamento de redes através de um conjunto de softwares desenvolvidos pela equipe, que contam inclusive com o próprio sistema de criptografia e segurança interna. “Lazy Moon” visa apresentar uma maneira mais prática, rápida e fácil de se fazer atividades de um monitor de rede. O sistema completo inclui:


  LM - Login

✩ Sistema de login  

  LM - Management

✩ Gerenciamento de usuários do programa  

  LM - CService

✩ Configuração automatizada de serviços em servidores Linux  

  LM - MRI

✩ Monitoramento de conexão de dispositivos  

  LM - NetChat

✩ Chat para suporte  

  LM - SSHost

✩ Acesso remoto a computadores Linux  

  Relatórios

✩ Relatórios de uso de hardware de servidores  

  Lazy Moon visa apresentar uma maneira mais prática, rápida e fácil de se fazer atividades de um administrador de rede.
