Lazy Moon

Projeto

  O projeto "Lazy Moon" tem como objetivo criar um sistema de monitoramento e suporte remoto a uma rede LAN, utilizando-se de diversos programas criados pelos integrantes. Os programas, que funcionam de maneira interligada, proveem funcionalidades como:

✩ Sistema de login; 
✩ Gerenciamento de usuários do programa; 
✩ Monitoramento de conexão de dispositivos; 
✩ Relatório de uso de hardware de servidores; 
✩ Chat para suporte; 
✩ Acesso remoto a computadores Linux; 
✩ Configuração de serviços em servidores Linux automatizados. 

  Lazy Moon visa apresentar uma maneira mais prática, rápida e fácil de se fazer atividades de um administrador de rede.
